# Vijeta Design System
Version 1.0

---

# Design Philosophy

Vijeta is not a job portal.

It is not another hackathon listing website.

It should feel like the operating system for ambitious students.

The UI should communicate

- Premium
- Friendly
- Youthful
- Trustworthy
- Motivating

Think

Linear × Framer × Notion

instead of

Government Portal × College ERP.

---

# Design Inspiration

Reference

https://sienna-popup-358899.framer.app/

Use this landing page as the visual benchmark.

Maintain

- Layout
- Section spacing
- Hero composition
- Typography hierarchy
- Card design
- Navigation
- Animation philosophy
- Clean minimal aesthetic

Do NOT copy branding.

Replace all visuals and messaging with Vijeta branding.
